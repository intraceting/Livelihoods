Introduction

This page is for distributing information to the (ad hoc) CIFS Working Group that isn't yet ready to be posted to the official CIFS homepage. CIFS (Common Internet File System) is a file sharing protocol. CIFS was formerly known as SMB (Server Message Block), the file sharing protocol used by MS-Net, LAN Manager, and Windows Networking. In addition, there are SMB solutions for OS/2, Netware, VMS, and Unix from vendors such as AT&T, HP, SCO, and, via Samba, over 33 others.

The WG hangs out at:

  • The CIFS@discuss.microsoft.com mailing list (to subscribe, e-mail LISTSERV@DISCUSS.MICROSOFT.COM with a message of: "SUBSCRIBE CIFS"; to unsubscribe, send "SIGNOFF CIFS")
    • The archives for the list are at: http://discuss.microsoft.com/archives/index.html
    • The list administrator is CIFS-request@DISCUSS.MICROSOFT.COM
    • See http://www.microsoft.com/sitebuilder/resource/mailfaq.asp for more information about Microsoft mailing lists

The Official homepage for Microsoft CIFS products has moved to: http://www.microsoft.com/intdev/cifs. This page assumes you've got past all that and are ready for more.
